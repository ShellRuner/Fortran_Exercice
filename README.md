Project : Fortran Exercice

This project was a school assignment consisting of several Exercices to be solved gradually


To execute the code you need to:
   - Install the fortran compiler : gfortran

To execute the differente Exercices code in your terminal:
  - Use [gfortran filename.f95]
  - then execute the command [./a.out] or [./a.exe]
  
  !!NB: Some file could take time to execute
